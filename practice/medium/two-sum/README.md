# Two Sum

## Difficulty: Medium

## Description
### Description
Given an array of integers `nums` and an integer `target`, return the 0-based indices of the two numbers such that they add up to `target`.

You may assume that each input would have exactly one solution, and you may not use the same element twice.

### Input Format
- The first line contains two integers: **N** (the number of elements in the array) and **Target**.
- The second line contains **N** space-separated integers representing the array `nums`.

### Output Format
Print the two indices on a single line separated by a single space, in ascending order.

### Constraints
* $2 le N le 10^4$
* $-10^9 le \text{nums}[i] le 10^9$
* $-10^9 le \text{Target} le 10^9$

### Examples
**Example 1:**
Input:
```
4 9
2 7 11 15
```
Output:
```
0 1
```
Explanation: Because nums[0] + nums[1] == 2 + 7 == 9, we print "0 1".

**Example 2:**
Input:
```
3 6
3 2 4
```
Output:
```
1 2
```
